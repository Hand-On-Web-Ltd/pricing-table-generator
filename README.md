# Pricing Table Generator

Build good-looking pricing tables without writing code. Add tiers, set prices, pick a style, and export the HTML when you're happy with it.

Pure HTML, CSS, and JavaScript — no dependencies, no build step.

## Features

- **Add/remove tiers** — Create as many pricing columns as you need
- **Set details** — Name, price, feature list, and CTA button text per tier
- **Style picker** — Choose from rounded, flat, or gradient themes
- **Monthly/annual toggle** — Switch between billing periods
- **HTML export** — Copy the generated embed code and paste it into any website
- **Live preview** — See changes as you make them

## Usage

1. Open `index.html` in your browser
2. Use the editor panel to add pricing tiers
3. Fill in names, prices, and features
4. Pick a style from the dropdown
5. Click "Export HTML" to copy the embed code

## Use Cases

- SaaS product pages
- Agency service packages
- Freelancer rate cards
- Course or membership pricing
